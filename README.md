# ClarifyQA 🔍

> ClarifyQA is an AI reasoning agent that helps QA Engineers analyze unclear requirements, detect hidden risks, ask better questions, and create smarter test coverage before testing starts.

Built for the **Microsoft Agents League Hackathon 2026** using Microsoft Foundry and Python.

---

## The Problem

QA Engineers often receive unclear, incomplete, or untestable requirements — leading to:
- Missed bugs
- Weak test coverage
- Repeated clarification meetings
- Costly rework

## The Solution

ClarifyQA helps QA Engineers move from **reactive testing to proactive quality analysis**.

---

## How It Works

ClarifyQA uses a multi-agent architecture with three specialized agents:

| Agent | Role |
|-------|------|
| **Analyzer Agent** | Checks requirement clarity, completeness, and testability. Detects gaps. |
| **Strategist Agent** | Generates clarification questions and risk-based test plan |
| **Generator Agent** | Creates test scenarios and professional bug reports |
| **Orchestrator** | Coordinates all three agents based on user input |
